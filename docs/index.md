# Media Server Documentation

Setting up a proper linux server to host a media library can be quite time consuming. Even with modern tools this can be rather cumbersome. This is why I created this documentation to set up everything.

## Arr Apps and others

Configuration guides for Sonarr, Radarr, Seerr, NZBGet, Unpackerr and Plex in detail are not provided yet. Since user preferences differ for these tools this has rather low priority but guides for that are nonetheless in progress.

## Work in Progress

This documentation is by no means final and can always be improved further.
Security has been taken into consideration but it is still expected to make your own thoughts on why we are doing something and how to properly do something.
