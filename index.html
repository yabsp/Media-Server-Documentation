    Media Server Documentation

    Setting up a proper linux server to host a media library can be quite time consuming - even when using modern tools. This is why I created this documentation.

    Arr Apps and others

    Configuration guides for Sonarr, Radarr, Seerr, NZBGet, Unpackerr and Plex in detail are not provided yet. Since user preferences differ for these tools, priority is rather low for these guides - nonetheless this is planned for future updates.

    Work in Progress

    This documentation is by no means final and can always be improved further. Security has been taken into consideration but there might still be risks and issues with this setup. It is expected to make your own thoughts on why we are doing something and how to properly do something.
